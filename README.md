Movie Rental System

The Movie Rental System is a backend REST API built using Node.js, Express, MongoDB, and Mongoose.  
It allows users to manage movies, users, rentals, and reviews in a structured and scalable system.

The system supports full CRUD operations, movie search with pagination, and advanced analytics using MongoDB aggregation pipelines.

-TECHNOLOGIES USED
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman (for API testing)
- Git & GitHub
- IntelliJ IDEA

-FEATURES

users:

- Create user
- Get all users
- Get user by ID


Movies
- Add, update, delete movies
- Get all movies
- Get movie by ID
- Search movies with pagination

Rentals
- Rent a movie
- Return a movie
- Automatic inventory management (available copies update)

Reviews
- Add review
- Get all reviews

Analytics
- Top-rated movies
- Most active users
